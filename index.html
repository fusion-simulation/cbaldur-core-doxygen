cbaldur

CBaldur 是一个现代化磁约束聚变堆集成模拟软件，采用 C++ 语言编写，集成了等离子体平衡求解、输运模拟、波加热与驱动、杂质辐射等多个模块，旨在为磁约束聚变堆的设计与优化提供高效、准确的数值工具。

参与开发

  1. 克隆代码库：
git clone https://github.com/fusion-simulation/cbaldur-core.git
  1. 编译、测试
cd cbaldur-core
mkdir build && cd build
cmake ..
make
ctest
  1. 提交代码：创建分支、提交更改、发起拉取请求
git checkout -b feature/your-feature-name
git add .
git commit -m "Add your commit message"
git push origin feature/your-feature-name
