详细描述

Input parameters and configuration structures for the simulation. This file defines various structures that hold configuration parameters and input data for the simulation. Each structure corresponds to a specific aspect of the simulation's configuration.

  • RunControl: General run control parameters.
  • Switches: Various switches to control simulation features.
  • RFHeating: Parameters related to radio frequency heating.
  • RadiationAndReactionsRates: Parameters for radiation and reaction rates.
  • NeutralGas: Parameters for neutral gas modeling.
  • NeutralBeam: Parameters for neutral beam injection.
  • AlphaParticles: Parameters for alpha particle modeling.
  • CBaldEquilibrium : Parameters for 2D equilibrium configuration.
  • MHDEquilibrium: Parameters for MHD equilibrium and stability calculations.
  • InitialBC: Initial and boundary conditions for the simulation.
  • PelletInjection: Parameters for pellet injection modeling.
  • TransportModels: Parameters for transport models used in the simulation.
  • OtherAuxHeating: Parameters for other auxiliary heating methods.
  • Numerics: Numerical parameters and tolerances.
  • OutputControl: Parameters controlling output and logging.
  • DevAndDebug: Development and debugging parameters.
